# Amazon Sales Dashboard - MS Excel Project

## 📌 Project Overview

This project is an interactive **Amazon Sales Dashboard** created using **Microsoft Excel**. The dashboard provides insights into sales performance, customer behavior, product categories, and key business metrics through dynamic visualizations and data analysis techniques.

The objective of this project is to transform raw sales data into meaningful business insights that can support data-driven decision-making.

## 🚀 Features

* Interactive Dashboard with slicers and filters
* Sales Performance Analysis
* Category-wise and Product-wise Sales Insights
* Monthly and Yearly Sales Trends
* Top Performing Products Identification
* Customer and Order Analysis
* KPI Metrics Visualization
* Clean and User-Friendly Layout

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning
* Data Validation
* Excel Formulas and Functions

## 📊 Key Insights

* Identification of top-selling product categories
* Analysis of sales trends across different periods
* Revenue contribution by various segments
* Order distribution and performance tracking
* Business performance monitoring through KPIs

## 📂 Project Structure

* Raw Data Sheet
* Data Cleaning Sheet
* Pivot Tables
* Dashboard Sheet
